# Usage

```java

```

or

```java
[DiagramConnector <value> =]
DIAGRAM::AddConnector(fromShape=<fromShape_path>,toShape=<toShape_path>)
```

where

-\<fromShape_path\> addresses a `DiagramShape`
-\<toShape_path\> addresses a `DiagramShape`
