# <tt>AddConnector</tt> &nbsp; <img src="./images/AddConnector_BIG.png" alt="AddConnector"/>

[//]: # (Do not edit this file, which is automatically generated)

This edition primitive addresses the creation of a new connector linking two shapes in a diagram

---

## Usage

```java
[DiagramConnector <value> =]
AddConnector(toShape=<toShape_path>,fromShape=<fromShape_path>)
```

or

```java
[DiagramConnector <value> =]
DIAGRAM::AddConnector(toShape=<toShape_path>,fromShape=<fromShape_path>)
```

where

- \<toShape_path\> addresses a `DiagramShape`
- \<fromShape_path\> addresses a `DiagramShape`

---

## Configuration

| Property        | Type                    | &nbsp;Required&nbsp;  |
| --------------- |-------------------------| :------:|
| `toShape` &nbsp; | `DiagramShape` &nbsp; | yes |
| `fromShape` &nbsp; | `DiagramShape` &nbsp; | yes |

- `fromShape` : 

## Examples

```java
myConnector = AddConnector(
        fromShape=myFirstShape,
        toShape=mySecondShape) in (myDiagram);
```

Creates a new connector in Diagram identified by ‘myDiagram’ expression, binding the shapes respectively identified by 'myFirstShape' and 'mySecondShape', and assign this new connector to ‘myConnector’
