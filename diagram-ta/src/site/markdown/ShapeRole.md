## Usage

```java
[visibility] [cardinality] DiagramShape <identifier>
with ShapeRole(metamodelElement=<metamodelElement_path>[,options]);
```

or

```java
[visibility] [cardinality] DiagramShape <identifier>
with DIAGRAM::ShapeRole(metamodelElement=<metamodelElement_path>[,options]);
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
- \<metamodelElement_path\> addresses a `T`
